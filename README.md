# SVD 圖像壓縮

線性代數課程作業:利用奇異值分解(SVD)實作圖像壓縮。

# 原理

對影像矩陣 A 進行奇異值分解:

$$A = U \Sigma V^T$$

奇異值按大小排序,前面少數幾個就承載了影像大部分的結構資訊。因此只保留前 k 個奇異值即可近似重建影像:

$$A_k = U_k \Sigma_k V_k^T$$

儲存量從原本的 `m × n` 降為 `k × (m + n + 1)`,當 k 遠小於 min(m, n) 時可大幅壓縮資料量。k 越小壓縮率越高,但畫質也越差。彩色圖片則對 B、G、R 三個通道分別做 SVD 再合併。


## 主要內容

- 對彩色圖片(BGR 三通道)進行 SVD 分解,保留前 k 個奇異值重建影像
- 比較不同 k 值(544 / 100 / 50 / 25)的畫質變化
- 計算 MSE 與 PSNR 進行品質定量評估,並分析壓縮比與壓縮率
- 使用 OpenCV Trackbar 實作互動式滑桿,即時預覽不同 k 值的壓縮效果
